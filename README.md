# ng-budget

A personal budget tracking application built with Angular and PocketBase.

## About

This is a hobby project designed to track income and expenses across different budget categories. The app provides budget management, reporting, and expense tracking capabilities.

**Note:** This application is designed for single-user use only and is not intended for multi-user environments.

## Tech Stack

- Angular 21
- Angular Material
- PocketBase (backend/database)
- TypeScript
- Signals for state management

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm start

# Build for production
pnpm build
```
